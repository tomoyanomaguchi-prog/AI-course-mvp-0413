# 規模と負担方針（決定・要確認）

**目的**: 募集・予算・ツール・ハイブリッドの**意思決定メモ**。数値は開催前に再確認する。

**更新日**: 2026-04-11

---

## 1. 規模

| 項目 | 方針 |
|------|------|
| 募集上限 | **最大 40 名**（会場キャパ・運営シフトと整合） |
| 受講料 | **完全無料**（スポンサー・社内予算で賄う前提） |

---

## 2. オフライン・運営リソース（追記）

| 項目 | 方針 |
|------|------|
| 会場 | **既に押さえ済み**（詳細は運営内メモ）。**Day1・Day7・Day13（まくり）・Day14 のオフライン回は同一会場**でよい（インタビュー反映）。 |
| 社員シフト | **2 名分はシフト可能**（オフライン回の密度・受付・安全に充当） |
| 体感 | 上記により **40 名規模は運営上まわせる見込み**（メンター委託・OH とあわせて [`MENTOR_ONBOARDING.md`](MENTOR_ONBOARDING.md) で監視） |

---

## 3. インフラ（標準レーン）・実費・端末

| 項目 | 方針 |
|------|------|
| GitHub / Vercel / Supabase 等 | **原則は無料枠のみ**（独自ドメイン・有料 API 等も**原則なし**）。逸脱は「上級者レーン・自己責任」と明記（[`STACK_CHOICE.md`](STACK_CHOICE.md) と整合） |
| **例外** | 有料利用が必要な場合は **社員が許可**した場合のみ。**申請は Google フォーム**（[`EXCEPTION_REQUEST_FORM_SPEC.md`](EXCEPTION_REQUEST_FORM_SPEC.md)）。**SLA: 3営業日以内**に返信 |
| **学生プラン対象外（Cursor Pro 等）** | **Plan B は自費**で Pro 等に加入してもらう（FAQ に明記。[Cursor 学生](https://cursor.com/ja/students) は**開講前に再確認**） |
| **端末** | ノート PC 推奨。**PC が不調な場合はレンタル等も検討**（**予算を検討するときに台数・単価を一緒に決める**） |
