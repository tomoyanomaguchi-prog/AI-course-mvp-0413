# 参加者規約（たたき台）

**目的**: **申込フォームでチェック必須**にするための骨子（[`COMMUNITY_AND_COMMS.md`](COMMUNITY_AND_COMMS.md)）。

**本文の作り方**: **過去に作った有料コース用の規約ひな型**をベースに、本プログラム向けに修正する。**主催が社内法務に確認**してから確定文へ差し替える。

---

## 含める項目（ドラフト見出し）

1. プログラムの目的・期間・無料であること  
2. **ハラスメント禁止**・**個人情報**（連絡先の利用目的）  
3. **秘密情報**（API キー等を公開リポジトリに含めないことの理解）  
4. **未成年**が参加しうること・**保護者同意は求めない**方針（本プログラムの運用上の理由）  
5. **写真・SNS**（集合写真の掲出、写りたくない場合は申し出）  
6. **録画なし**（原則）  
7. **免責**（サービス提供の範囲、PC 故障等は原則自己責任の枠組み）  
8. **準拠法・管轄**（要法務）
