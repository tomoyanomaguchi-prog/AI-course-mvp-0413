# コミュニケーション・Discord・SNS（運営方針）

**目的**: 受講生・メンター・主催の**連絡の正**と、**質問の集約**、**発信の習慣**を一文にまとめる。

---

## 1. Discord（サーバー・ロール・チャンネル）

| 項目 | 方針 |
|------|------|
| **サーバー** | **主催がオーナー**で固定（インタビュー反映） |
| **ロール** | **3 つ**で運用（例: **受講生**／**メンター**／**運営・主催** — 名称は実装時に確定） |
| **受講生のメイン** | **Discord**（第1期の正。以降も原則 Discord） |
| **質問・チケット** | **公開チャンネルは 1 本**に集約（例: `#質問` / `#support` など **1 つだけ**） |
| **例外** | **メンターへの DM も可**だが **推奨しない**（検索性・公平性・抜け漏れ防止のため） |
| **招待リンク** | **期限なし**でよい（インタビュー反映。Discord の設定で **失効なし**） |
| **参加の案内** | **Day0 で**サーバー参加・チャンネル説明をする（**募集〜Day0 前に Discord 必須とはしない**） |
| **期間終了後** | **コホート終了後**に **受講生ロールの整理・キック**（途中退講者も **原則は期の終了後**にまとめて扱う。インタビュー反映） |

詳細フォーマットは [`mentor_ops_handbook.md`](mentor_ops_handbook.md)（Discord に合わせて文言調整）。**チャンネル構成・表示名・命名規則**は [`DISCORD_NAMING_AND_CHANNELS.md`](DISCORD_NAMING_AND_CHANNELS.md)。

## 2. SNS・発信（積極化）

**方針**: **むしろ積極的に投稿してほしい**（インタビュー反映）。

| 項目 | 方針 |
|------|------|
| **必須フォロー** | **Instagram [@geeksalon](https://www.instagram.com/geeksalon/)**（詳細は [`SNS_ASSETS_AND_TEMPLATES.md`](SNS_ASSETS_AND_TEMPLATES.md)） |
| **ハッシュタグ** | **`#geeksalon`**（固定1本。同上） |
| **タイミング** | **Day0 以降**で **フォロー・ハッシュタグ**を案内してもよい（**Discord と同様**） |
| **運用** | **Day0 は説明のみ**のため **公式の投稿テンプレ運用は必須にしない**。**Day1 キックオフ・Day7 中間報告・Day14 最終発表**の **3 回**を [`SNS_ASSETS_AND_TEMPLATES.md`](SNS_ASSETS_AND_TEMPLATES.md) で想定。文言は **当日・直前**に配布してよい |
| **注意** | 個人が特定される内容・他者の写り込みは **本人の判断**。**集合写真**は [`LP_AND_SPONSOR_RULES.md`](LP_AND_SPONSOR_RULES.md) と整合 |

**補足（「オプトイン署名」とは）**: 集合写真に写ることへの**書面同意**を一人ひとつ取る方式。本プログラムでは **採用しない**。**集合写真は通常どおり掲出**し、**写りたくない人は当日に申し出てほしい**程度の運用（インタビュー反映）。

**テンプレート・ロゴ入手**は [`SNS_ASSETS_AND_TEMPLATES.md`](SNS_ASSETS_AND_TEMPLATES.md)。

## 4. 未成年・保護者同意
