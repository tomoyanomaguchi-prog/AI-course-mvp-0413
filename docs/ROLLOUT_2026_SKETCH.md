## エントリーと LP（計測・コラボ）

「エントリー」は **募集の申込**に限らず、**Day0・Day1 への参加導線**も含む。**各日程で LP を分ける**とよい（詳細は [`LP_AND_SPONSOR_RULES.md`](LP_AND_SPONSOR_RULES.md)）。

| 項目 | 方針 |
|------|------|
| **募集 LP** | **Encourage** 上。**募集開始（4/15）に合わせる**。**1ページ**でフォーム完結 |
| **計測** | **フォーム送信**に加え、**各 LP の表示・クリック**を **社内ツール**で取得（**募集／Day0／Day1 でイベントを分ける**） |
| **先着の正** | **フォームのタイムスタンプ**（[`LP_AND_SPONSOR_RULES.md`](LP_AND_SPONSOR_RULES.md)） |
| **先着の運用** | **手動**でコホート振り分け（スプレッドシート等） |
| 補足 | LP 外からの流入を許す場合は、**同一フォームに誘導**し LP 経由と揃えるか、運営で別途ルールを決める |

- **スプレッドシート・CSV 転記**は **日次が理想**。**逃したときの最低ライン**: **遅くとも翌営業日終了まで**に追いつく。**それも難しい週は週末に差分を一括**で埋める（インタビュー反映）。**当面は主催**。将来的に **AI 支援・派遣** を検討。
